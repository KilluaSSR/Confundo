# Confundo

> **我说不要在我的设备上做用户画像了，你尓多隆吗？**

## 简介

- 明明已经关掉了大多数权限了，在应用 A 中搜索的东西又到应用 B 里推荐了？它们到底是怎么识别出你的？答案是设备指纹。

- 利用系统 API，可以获得许多设备信息。这些 API 并不受权限系统管理，你也没办法拒绝。尽管这些设备信息单独看没有什么特别的，但是把它们组合在一起，就足以识别出你的设备。
    - 你觉得天下有多少台 设备地区是 en_US，时区是 Asia/Shanghai，首次激活时间是2026年5月1日14点15分16秒，打开了高对比度文字，Kernel版本是6.6.XXXX，最小宽度是447dp，关闭了24小时制显示的三星 S25Ultra？

### 功能

**Introducing The Confundus Charm - 混淆咒**

- 一键给**所有应用**都生成随机设备数据，**且各个应用互不相同**。这样，它们读出的数据不同，大大增加了设备指纹识别的难度。
- 定时随机刷新设备数据。

### 使用方式

1. 安装 LSPosed。在 LSPosed 启用 Confundo，勾选所有你想要伪装的 App。
2. 在 Confundo 中启用所有你想要伪装的 App，并给它随机生成一套模板或手动编写每个字段。
3. 即时生效。
